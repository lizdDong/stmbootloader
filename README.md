# stmbootloader

#### 介绍
支持Ymodem模式传输升级，支持从内部flash升级

#### 使用说明
```
============ STM Bootloader =============
 Author:    lizdDong                     
 Revision:  V1.1 @2021-7-1               
 Application address: 0x08004000             
 Image       address: 0x0801C000             
 Auto run application after 1(s).       
 Key <F1>  run application.              
 Key <F2>  upgrede via Ymodem.           
 Key <F3>  forced to upgrede from image! 
=========================================
```
